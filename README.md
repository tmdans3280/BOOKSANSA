# BookMovie

책을 영화처럼 추천받고, 즐겨찾고, 리뷰까지 볼 수 있는 도서 기반 웹앱입니다.  
React + Kakao Book API 기반으로 제작되었으며, 검색/찜하기/상세보기 기능을 제공합니다.

---

## ✨ 주요 기능

- 실시간 검색 (Debounce 적용)
- 도서 상세 정보 보기 (썸네일, 저자, 출판사, 출간일 등)
- 찜하기 기능 (localStorage 기반)
- 반응형 UI (모바일, 태블릿 대응)
- 모달 기반 도서 미리보기
- Swiper 슬라이드 구성 (신간 추천 섹션)

---

## 🔧 사용 기술

- React (Vite)
- Tailwind CSS
- React Router v6
- Kakao Book API
- React Icons
- Swiper.js
